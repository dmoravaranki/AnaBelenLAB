# BeeUI Component Patterns

## Stat Card
Use `.bee-stat-card`, `.bee-stat-value`, `.bee-stat-label`, `.bee-stat-change`.

## Product Card
Use `.bee-product-card`, `.bee-product-image`, `.bee-product-body`, `.bee-product-title`, `.bee-product-price`.

## Tabs
Use `.bee-tabs`, `.bee-tab`, `.bee-tab-active`.
